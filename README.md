## 项目简介

​    基于django的个人博客，比较简单。



#### 功能

- 用户注册（含验证码）、登录
- 文章发布（含富文本编辑器）
- 个人信息后台管理，密码修改，文章删除
- 文章列表展示，分页
- 文章详情页（含评论功能）
